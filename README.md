# RFID Attendance System

An RFID-based attendance management system built using C++ and microcontroller integration. The system records attendance automatically by scanning RFID cards.

## Features
- RFID card scanning
- Automatic attendance marking
- Unique ID verification
- Fast and contactless system
- Simple and efficient tracking

## Technologies Used
- C++
- RFID Module
- Microcontroller Programming

## How It Works
1. RFID card is scanned
2. System reads unique card ID
3. ID is verified
4. Attendance is marked automatically
